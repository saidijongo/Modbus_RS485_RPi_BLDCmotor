# Modbus_RS485_RPi_BLDCmotor
Using RS485 serial communication for BLDC motor control in the raspberry pi.
 Used hardware:
 Raspbery pi 4 (CM4)
 Hall sensored BLDC Motor wheels
 BLDC Motor Driver
 Max485 module
